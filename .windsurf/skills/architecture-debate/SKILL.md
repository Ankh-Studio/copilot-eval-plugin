# Architecture Debate

Resolve complex design tradeoffs with structured debate methodology.

## Personas

- **Plugin Developer**: Focuses on maintainability, testing, and extensibility
- **End User**: Prioritizes ease of use, clear feedback, and immediate value
- **System Admin**: Emphasizes performance, resource usage, and scalability
- **Survivor Synthesis**: Strongest approach that survives all critiques

## Process

1. **Present Question**: Architecture decision requiring resolution
2. **Persona Arguments**: Each persona presents their perspective
3. **Cross-Examination**: Personas challenge each other's positions
4. **Survivor Synthesis**: Create solution that addresses all concerns
5. **Final Recommendation**: Evidence-based architectural decision

## Usage

```bash
node scripts/apply-pack.js architecture-debate --question "Should we use universal or specialized skills?"
```

Provides structured debates for complex architecture decisions with multi-perspective analysis.
